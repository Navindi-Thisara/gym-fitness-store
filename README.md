## ⚙️ Setup Instructions

1. Clone the repository
2. Move to `htdocs`
3. Start XAMPP (Apache + MySQL)
4. Import `gym_store.sql`
5. Configure `config/db.php`
6. Run:

```
http://localhost/gym-store
```
